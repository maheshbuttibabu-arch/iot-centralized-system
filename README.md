# Centralized IoT System

## Description
This project demonstrates a centralized IoT system using MQTT and Node-RED.

## Features
- Home appliance control (Light ON/OFF)
- Robot control (Forward/Stop)
- Real-time monitoring
- Web and mobile dashboard

## Architecture
Node-RED acts as the master hub communicating with devices via MQTT.

## Setup Instructions
1. Install Node-RED
2. Import flow.json
3. Open dashboard at http://localhost:1880/ui

## Technologies Used
- Node-RED
- MQTT (HiveMQ public broker)
